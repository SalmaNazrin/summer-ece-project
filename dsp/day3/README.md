Digital Filtering
Designed and applied Butterworth low-pass and high-pass filters using SciPy to isolate individual frequency components from a noisy mixed signal. Used filtfilt() for zero-phase filtering to avoid signal distortion. Compared the original noisy signal against the filtered output to verify that the filters correctly removed unwanted frequencies.
Concepts covered: Butterworth filter, low-pass filter, high-pass filter, cutoff frequency, zero-phase filtering, normalized frequency
Libraries used: NumPy, SciPy, Matplotlib
